== Description ==

Send notifications whenever a post or page is edited/deleted to all users in a specified role.

Let me know of any problems: jj56@indiana.edu

== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Choose the role to send edit notifications to in the admin panel
